# 아케인 샌드박스 (Arcane Sandbox)

**"마법을 과학처럼 연구하라."**

아케인 샌드박스는 플레이어가 고문헌을 해독하고, 직접 캔버스에 마법진과 룬 문자를 그려가며 미지의 마법 법칙을 발견하는 고자유도 퍼즐/시뮬레이션 웹 게임입니다.

---

## 🔮 게임의 핵심 비전 (Core Vision)

단순히 정해진 아이콘을 배치하거나 클릭하는 상투적인 시스템을 탈피했습니다. 마법진을 그리는 **필기 행위 그 자체**가 물리 엔진이자 마법 연산의 핵심이 됩니다.

1. **도형 뼈대와 룬 문자 결합 시스템**
   - 기하학적 뼈대(원, 다각형 등)를 그리고, 그 위에 마우스를 드래그하여 직접 룬 문자를 필기합니다.
2. **형태학 및 속도 엔진 (Stroke & Dynamics)**
   - 플레이어가 그린 선(Stroke)을 시스템이 분석합니다.
   - **결손/확장**: 정석적인 문자에서 획 하나를 덜 그리면 마법이 불안정해지고, 선을 하나 덧대면 파생 마법(부메랑 효과 등)이 생깁니다.
   - **필압/속도**: 글씨를 천천히 정성껏 쓰면 지속형 마법이, 빠르고 거칠게 휘갈겨 쓰면 폭발형 마법이 발현됩니다.
3. **부수 조립 (Radical Combinations)**
   - 룬 언어는 한글/한자처럼 부수의 결합입니다. '열기(△)'와 '대지(ㅡ)' 부수를 직접 겹쳐 그려 새로운 '마그마' 룬을 창조할 수 있습니다.
4. **고문헌 아카이브 (Archives)**
   - 스킬 트리가 아닌, 순차적으로 주어지는 고대 문서의 모호한 스케치와 힌트를 해독하고 캔버스에서 이를 재현하여 문법을 깨우쳐야 합니다.

## 🌌 메타 게임플레이 (Meta Gameplay)

- **차원 균열 (Abyssal Rift)**
  화면 중앙의 미지의 균열에 완성된 마법을 주입하여 반응을 관측하고, 캔버스를 오염시키는 균열의 '간섭파'를 방어합니다.
- **캔버스 흉터 시스템 (Scarred Canvas)**
  마법진이 한계를 넘어 폭주하면, 실험 캔버스의 일부가 영구적으로 타버려 오염된 공간이 됩니다.
- **마도 공학 자동화 (Factorio-style Automation)**
  성공적인 마법진 패턴을 모듈화하고, 수십 개의 모듈을 마력선으로 이어 붙여 거대한 자동화 마법 회로를 설계합니다.

## 🛠️ 기술 스택 (Tech Stack)

- **Frontend**: Vanilla JavaScript (HTML5, CSS3)
- **Build Tool**: Vite
- **Graphics**: HTML5 `<canvas>` API (실시간 드로잉, 선 분석 및 렌더링)
- **UI Design**: Arcane Punk (Dark Mode + Neon Glow + Glassmorphism)

## 🚀 시작하기 (Getting Started)

1. 저장소를 클론하거나 다운로드합니다.
2. 의존성 패키지를 설치합니다.
   ```bash
   npm install
   ```
3. 로컬 개발 서버를 실행합니다.
   ```bash
   npm run dev
   ```
4. 브라우저에서 `http://localhost:5173` 에 접속하여 캔버스 드로잉 테스트를 시작합니다.
